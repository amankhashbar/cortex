Build the sensor suite

From parts to a verified live stream.

Build and verify one sensor at a time. Start with fingertip PPG over USB; add the motion sensor after the pulse stream is stable.

01

Gather the core parts

ESP32 DevKit V1, one MAX30102 PPG module, one MPU6050 motion sensor, jumper wires, a breadboard or female-to-female Dupont wires, and a USB data cable.

Begin with one MAX30102. Two MAX30102 modules share address 0x57 and need a multiplexer or separate I²C bus.

02

Wire the shared I²C bus

Sensor pinESP32 pin
MAX30102 VIN3V3
MAX30102 GNDGND
MAX30102 SDAG21
MAX30102 SCLG22
MPU6050 VCC / GND3V3 / GND
MPU6050 SDA / SCLG21 / G22

Disconnect USB power before changing wires. Use 3.3 V, not the ESP32 5 V pin.

03

Install and flash the firmware

  1. Install Arduino IDE and Espressif's ESP32 board package.
  2. Install the SparkFun MAX3010x and Adafruit MPU6050 libraries.
  3. Open firmware/neuroreadiness.ino.
  4. Select ESP32 Dev Module, choose the USB serial port, and upload.
04

Verify before testing

At 115200 baud, the serial stream should begin with # NRDY,1, followed by rows in this format:

t_ms,red,ir,ax,ay,az

Cover the MAX30102 with a fingertip and confirm that red and IR readings respond before starting the diagnostic.

Ready to connect

Launch the live diagnostic

Use desktop Chrome or Edge over HTTPS. The browser will ask you to select the ESP32 serial port.

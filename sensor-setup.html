Build the sensor suite

From parts to a verified live stream.

Build and verify one sensor at a time. Start with fingertip PPG over USB; add the motion sensor after the pulse stream is stable, then add the CJMCU-6701 GSR module for electrodermal arousal context.

01

Gather the core parts

ESP32 DevKit V1, one MAX30102 PPG module, one MPU6050 motion sensor, one CJMCU-6701 GSR skin-resistance module, jumper wires, a breadboard or female-to-female Dupont wires, and a USB data cable.

Begin with one MAX30102. Two MAX30102 modules share address 0x57 and need a multiplexer or separate I²C bus.

02

Wire the shared I²C bus and GSR analog pin

Sensor pinESP32 pin
MAX30102 VIN3V3
MAX30102 GNDGND

MPU6050 VCC / GND3V3 / GND
MPU6050 SDA / SCLG21 / G22
CJMCU-6701 VCC / GND3V3 / GND
CJMCU-6701 SIGG34

Disconnect USB power before changing wires. Use 3.3 V, not the ESP32 5 V pin. GSR is an arousal proxy only; it is not an emotion, lie-detection or diagnostic sensor.

03

Install and flash the firmware

  1. Install Arduino IDE and Espressif's ESP32 board package.
  2. Install the SparkFun MAX3010x and Adafruit MPU6050 libraries. The CJMCU-6701 uses a plain ESP32 analog input and needs no Arduino library.
  3. Open firmware/neuroreadiness.ino.
  4. Select ESP32 Dev Module, choose the USB serial port, and upload.
04

Verify before testing

At 115200 baud, the serial stream should begin with # NRDY,2, followed by rows in this format:

t_ms,red,ir,ax,ay,az,gsr

Cover the MAX30102 with a fingertip and confirm that red and IR readings respond. Touch the GSR electrodes and confirm the final value changes before starting the diagnostic.

Ready to connect

Launch the live diagnostic

Use desktop Chrome or Edge over HTTPS. The browser will ask you to select the ESP32 serial port.
